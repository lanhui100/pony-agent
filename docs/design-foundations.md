# 第一阶段视觉基础规范

## 风格定位

Pony Agent 当前采用“研究工作台”方向：

- 关键词：克制、理性、清晰、可学习
- 视觉气质：黑白灰主导，少量暖橙点亮
- 使用场景：桌面 agent 工作台、状态面板、执行链路展示、学习辅助说明

## 色彩基础

### 中性色

- 主文字：`#1F1F1B`
- 次文字：`#4F4A44`
- 辅助文字：`#666159`
- 弱提示文字：`#78736C`
- 页面背景亮部：`#F5F4F0`
- 页面背景过渡：`#F0EEEA`
- 页面背景深部：`#EBE8E2`
- 白色面层：`rgba(255, 255, 255, 0.58 - 0.78)`

### 强调色

- 主强调暖橙：`#C67938`
- 柔和暖橙氛围：`rgba(208, 124, 54, 0.13)`
- 成功态偏暖棕：`#6F4322`
- 错误态偏赤棕：`#8B3D2F`

规则：

- 暖橙只用于状态点、关键动作、少量提示
- 不把暖橙大面积铺成背景
- 大部分区域仍应由黑白灰完成层次建立

## 圆角体系

- 主面板：`8px`
- 次级卡片：`6px`
- 按钮：`4px`
- 对话气泡：`6px`
- 不使用大圆角和胶囊化面板语言

## 层次体系

优先顺序：

1. 面积与布局主次
2. 字号与字重
3. 明度层次
4. 少量强调色

避免：

- 重阴影
- 粗边框
- 同时出现过多视觉强调点

## 排版建议

- 标题：偏高字重，强调结构
- 正文：中等字重，优先可读性
- 辅助说明：颜色更浅，行高更宽松
- 学习说明与系统状态要有明显文案语气差异

## 组件原则

- 对话区始终是主体区
- 状态、架构、学习提示是辅助区
- 卡片只是信息容器，不应抢内容风头
- 图标用于扫读和定位，不用于装饰堆砌

## 第一阶段适用范围

这套规范优先用于：

- 首页工作台
- 对话区
- 运行状态区
- 执行步骤区
- 学习说明面板

后续如果进入更复杂的多面板或图谱阶段，可以在不破坏这套基础气质的前提下继续扩展。
